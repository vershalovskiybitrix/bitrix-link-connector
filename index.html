Ссылка на мессенджер

Выберите мессенджер, который нужно зарегистрировать как коннектор для Открытых линий. После регистрации он появится в настройках любой Открытой линии.

Инструкция:
1. Нажмите на мессенджер выше — он будет зарегистрирован как коннектор.
2. Перейдите в Контакт-центр → Открытые линии, выберите нужную линию.
3. В разделе коннекторов найдите зарегистрированный мессенджер и введите ссылку.
4. Добавьте Открытую линию в Виджет на сайт — кнопка появится автоматически.
Оставьте пустым, чтобы использовать иконку по умолчанию (🌐)
